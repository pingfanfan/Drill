## [1.0.0] - 2024-XX-XX

### Added
- Core SKILL.md with 5-layer framework
- Material collection tactics (automated + user-assisted)
- Question bank with domain-specific templates
- Output examples for various scenarios
- Limited-material analysis mode
- Troubleshooting guide for tool failures

### Features
- **Layer 1**: Raw battlefield material collection
- **Layer 2**: Unspoken consensus extraction
- **Layer 3**: Assumption deconstruction (3-assumption framework)
- **Layer 4**: Adversarial validation (5-challenge test)
- **Layer 5**: Quality question synthesis

- Product strategy validation
- Business model stress-testing
